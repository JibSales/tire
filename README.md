# Tire

Tire is a lightweight JavaScript library for modern browsers. The syntax is insperad from jQuery.

## Browser support

* Chrome
* Safari
* Internet Explorer 8
* Firefox 3.5
* Opera 10.5

## Build

    npm install -g grunt
    make
    
  On Windows
  
    npm install -g grunt
    grunt.cmd
    
## Test
